## ⚙️ Backend Implementation  

The backend server was developed using Flask and connected to a MySQL database.  
A table named **student** was created to store student records.

---

## 🧪 Testing

All CRUD APIs were tested using Postman.

Testing included:

- Creating a new student (POST)  
- Retrieving all students (GET)  
- Retrieving a student by ID (GET)  
- Updating student details (PUT)  
- Deleting a student (DELETE)  

Validation was also tested by sending incomplete or invalid data.

Screenshots of Postman responses and MySQL database table are included in the submission zip file.
